# 🔱 TRIDENT REPORTING SYSTEM

Every mission should produce a report.

Reports transform evidence into actionable intelligence.

---

# Purpose

Reporting communicates findings clearly, accurately, and professionally.

Reports should support decision making.

Not simply document activity.

---

# Reporting Philosophy

Evidence produces findings.

Findings produce recommendations.

Recommendations produce action.

---

Mission Brief

Executive Summary

# Findings

Each finding should contain:

Title

Severity

Confidence

Description

Evidence

Impact

Recommendation

References

---

# Severity

Use consistent severity levels.

Critical

High

Medium

Low

Informational

---

# Confidence

Every finding must declare confidence.

Confirmed

Likely

Possible

Unknown

---

# Reporting Rules

Separate facts from interpretation.

Separate evidence from opinion.

Never exaggerate severity.

Document uncertainty.

Support every recommendation.

Write for both technical and executive audiences when appropriate.

Good reports create trust.

Professional reports tell a story.
