# Purpose

Evidence is the foundation of trustworthy intelligence.

Every observation should be supported by evidence.

Evidence allows observations to be verified, reviewed, reproduced, and trusted.

Without evidence there is no reliable intelligence.

---

# Definition

Evidence is any verifiable artifact that supports an observation.

Evidence does not make conclusions.

Evidence preserves facts.

Analysis interprets those facts.

---

# Philosophy

Evidence should answer one question:

"How do we know this?"

If that question cannot be answered,

the observation should be reconsidered.

---

# Evidence Identity

Every Evidence record receives a permanent identifier.

Example

EVD-2026-000421

Identifiers never change.

Evidence may be superseded,

but historical evidence remains valuable.

---

# Evidence Attributes

Every Evidence record should contain:

Evidence Identifier

Title

Description

Source

Evidence Type

Collection Time

Collector

Mission Reference

Related Observation(s)

Integrity Information

Tags

---

# Evidence Integrity

Evidence should preserve:

Original format

Original timestamps

Original metadata

Chain of custody (when applicable)

Evidence should never be modified unnecessarily.

Copies may be created for analysis.

Original evidence should remain unchanged.

---

# Evidence Lifecycle

Evidence is:

Collected

↓

Verified

↓

Referenced

↓

Correlated

↓

Archived

Evidence should remain available after mission completion whenever legally and operationally appropriate.

---

# Evidence Relationships

Evidence supports:

Observations

Evidence may reference:

Entities

Missions

Techniques

Playbooks

Reports

Evidence should not directly create conclusions.

---

# Design Principles

Evidence should be:

Authentic

Repeatable

Traceable

Preserved

Organized

Technology-agnostic

Evidence should outlive the investigation.

---

# The Evidence Rule

Evidence answers:

"How do we know?"

It does not answer:

"What does it mean?"

Meaning belongs to analysis.

---

# Doctrine

Evidence preserves truth.

Observations describe truth.

Relationships organize truth.

Knowledge remembers truth.

Protect the evidence.

Protect the intelligence.

Observe.

Connect.

Remember.
