# History

## Age I — Threads: Genesis

The Watchers Platform began with a single idea:

Knowledge should not disappear.

Instead of building another AI assistant, the project set out to create a memory architecture capable of preserving, connecting, and compounding human knowledge.

This became known as **The Loom**.

The first four foundational models were defined:

- Thread
- Entity
- Relationship
- Event

Everything that followed was built upon them.
